# DEC-V61-094 · M-VIZ kickoff — 3D viewport infrastructure

## Why

Pivot Charter Addendum 3 §3.a (Accepted 2026-04-28 via DEC-V61-093) makes
3D viewport the product center. Addendum 3 §4.c HARD ORDERING requires
M-VIZ as the first milestone before M-RENDER-API / M-PANELS / M-AI-COPILOT.

This DEC is the M-VIZ kickoff — it does not implement the milestone, it
ratifies the implementation plan + scope + library choice + line-A/line-B
contract extension prerequisite + Path A engagement timeline. After
ratification, implementation work begins per the spec_v2 sequence.

Without this kickoff DEC ratified, no M-VIZ implementation commit lands —
the line-A contract extension prerequisite (V61-093 Kogami P2 #3) would be
violated and the library choice would be uncoordinated.

DEC(s) document the actual builds.
